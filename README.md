# XDKGame
**Demo: http://mmurp.com/cosmic-rising/**

A game made for the 2015 XDK game jam at Cal Poly.

In this game, players battle against spaceships in this 2D top down action game.

Controls
--------

* W - move up
* A - move left
* S - move down
* D - move right
* Left Mouse - fire weapon
* Right Mouse - move
* Scroll Wheel - zoom
